# 記事の追加・書き方マニュアル

このドキュメントでは、新しいレビュー記事を追加する手順を説明します。
特別なソフトは不要で、テキストエディタと画像があれば誰でも記事を追加できます。

## 1. 準備：画像の追加

記事で使う画像（サムネイルや本文中の画像）は、以下のフォルダに入れます。
`public/images`

推奨：フォルダがいっぱいにならないように、ファイル名はわかりやすくしましょう。
例: `rio_book_cover.jpg`, `sample_page_01.jpg`

## 2. 記事ファイルの作成

以下の場所に新しい **`.mdx`** ファイルを作成します。
`src/content`

**ファイル名のルール**:
- 半角英数字とハイフン(`-`)のみを使うのがおすすめです。
- このファイル名がそのままページの URL になります。
  - 例: `my-favorite-book.mdx` → `http://localhost:3000/blog/my-favorite-book`

## 3. 記事の書き方

作成した `.mdx` ファイルを開き、以下の形式で内容を書きます。
ファイルの一番上には、記事の情報を管理するための「設定エリア（フロントマター）」が必要です。

### 基本テンプレート

以下のテキストをコピーして、新しいファイルに貼り付けて書き換えてください。

```mdx
---
title: "作品のタイトル"
date: "2026-01-07"
rating: 5
thumbnail: "/images/your-image.jpg"
excerpt: "トップページに表示される短い紹介文をここに書きます。"
---

# 見出し1（大きな見出し）
ここに本文を書きます。
普通のテキストはそのまま表示されます。

*イタリック体（ジャンルやタグなどに使えます）*
**太字（強調したいところに使いましょう）**

## 見出し2（中くらいの見出し）
段落を変えるときは、1行空けてください。
（改行だけだとつながって表示されてしまいます）

ここに画像を貼りたいときは、通常のMarkdown記法ではなくテキストで説明するか、
Next.jsの機能を使う必要がありますが、まずはテキスト中心で書くのが簡単です。

> [**購入リンクのテキスト**](https://example.com)
```

### 設定項目の説明

ファイル冒頭の `---` で囲まれた部分の説明です。

| 項目 | 説明 | 例 |
| :--- | :--- | :--- |
| **title** | 記事のタイトル | `"素晴らしい同人誌"` |
| **date** | 投稿日 (YYYY-MM-DD形式) | `"2026-01-07"` |
| **rating** | 星評価 (1〜5の数字) | `5` |
| **thumbnail** | 作品の表紙画像など<br>`public`フォルダからのパス | `"/images/cover.jpg"` |
| **excerpt** | 一覧ページ用の短い説明 | `"最高の一冊でした。"` |

## 4. 便利な書き方テクニック

### 購入リンクを目立たせる
引用記法 `>` と太字 `**` とリンク `[]()` を組み合わせると、ボタンのように目立ちます。

```md
> [**メロンブックスで購入する**](ここにURL)
```

### キーワードを強調する
ジャンルや属性などを斜体（イタリック）で書くと雰囲気が出ます。

```md
*巨乳、幼馴染、純愛*
```

---

## 5. 確認方法

1. ターミナルで `npm run dev` を実行します（既に動いている場合はそのままでOK）。
2. ブラウザで `http://localhost:3000` を開きます。
3. 新しい記事がトップページに追加されているか確認しましょう！
